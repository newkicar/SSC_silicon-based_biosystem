## 提示词与文件名映射关系
**下表体现了用户的提示词对应的操作流程文件名称，你需要根据用户提供的信息，判断用户需要使用的流程文件，读取其名称，并在调用tool的时候将名称（带后缀）发送给tool**
| 提示词中关键词 | 操作流程文件名称         | skill文件路径           |
| ------------ -| ---------------------- | ----------------------- |
| 提交请假申请   | 提交系统请假申请.xlsx | skill-leave-request |



## 动态参数定义

**重要说明**：
- 所有参数名使用花括号括起来，如 `{leave_type}`
- 如果用户未提供参数值，**必须使用默认值**
- `skill_name` 是固定值，不需要从用户输入提取

### 参数表

| 参数名           | 类型   | 默认值                | 说明         | 提取规则                                                       |
| ---------------- | ------ | --------------------- | ------------ | -------------------------------------------------------------- |
| `skill_name`     | 固定值 | `skill-leave-request` | 技能目录名称 | 固定值，直接填入                                               |
| `{leave_type}`   | 字符串 | `年假`                | 假期名称     | 用户提到"请假类型XX"时提取；未提供时使用默认值                 |
| `{user_id}`      | 字符串 | `00110430`            | 用户ID       | 用户提到"IDXXX"时提取；未提供时使用默认值                      |
| `{start_time}`   | 字符串 | `2026-12-12 08:30`    | 开始时间     | 用户提到"开始时间XXX"时提取；未提供时使用默认值                |
| `{end_time}`     | 字符串 | `2026-12-12 17:00`    | 结束时间     | 用户提到"结束时间XXX"时提取；未提供时使用默认值                |
| `{leave_reason}` | 字符串 | `临时有事`            | 请假事由     | 用户提到"请假原因XXX"、"请假事由XXX"时提取；未提供时使用默认值 |
